## 一、江湖营销功能白皮书 (pageId=738306599) ✅

**负责人**: 王力  
**创建时间**: 2022-12-22  
**最后修改**: 2024-05-10  
**版本**: v1.2.0

### 7 种核心裂变玩法

| 玩法 | 拉新机制 | 关键设计 |
|------|---------|---------|
| **江湖优惠券** | 优惠券红包 | 私域提供社区活动玩法 |
| **裂变任务宝** | 邀请添加企微好友/注册微拍堂获取奖励 | 分享海报扫码添加江湖好友，活动引导通过企微回复实现 |
| **群裂变** | 邀请企微进群获取奖励 | 邀请企微进群/获取奖励 |
| **抽奖活动** | 大转盘，奖池随机概率中奖 | 任务宝、大转盘支持金豆作为奖品发放 |
| **金榜题名** | 答题，排行榜 top 获取奖励 | 答题排行榜 |
| **江湖集卡** | 完成任务获得碎片获取奖励，合成后获取大奖 | 完成任务获得碎片 |
| **江湖壕礼** | 邀请任务完成后邀请人和被邀请人同时获取奖励 | 双向奖励机制 |

### 通用规则
1. 江湖的活动只能在微信环境内打开和参与，不能在 app/小程序环境内参与
2. 参与用户必须为江湖企微好友，如果不是，提示加企微好友

### 数据权限
- 管理员可以看到并编辑所有成员创建的活动
- 成员只能查看并编辑自己创建的活动
